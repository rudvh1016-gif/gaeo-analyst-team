# 🏠 Gaeo 애널리스트팀 (KOSPI/KOSDAQ Home Office)

게임처럼 픽셀 캐릭터들이 집 안을 돌아다니며 주식을 분석하는 AI 애널리스트 팀.
**실시간 시세 + 실제 Claude 서브에이전트 분석**이 연결돼 있고, **매 거래일 자동 갱신**된다.

## 팀 구성 (5인 + 고양이)

| 캐릭터 | 방 | 역할 |
|---|---|---|
| **TARO** (헤드셋) | 거실 | 기술적 분석 — MA/RSI/MACD/거래량 |
| **FLOW** (초록 모자) | 거실 | 수급 분석 — 외국인·기관·개인 순매매 |
| **NOVA** (포니테일) | 부엌 | 뉴스·심리 분석 |
| **DIANA** (안경+긴머리) | 서재 | 재무·기본적 분석 — PER/PBR/ROE/실적 |
| **CHIEF** (수염+넥타이) | 침실 | 총괄 PM — 4인 종합 75% + 수급 25% → BUY/HOLD/SELL + 목표가·리포트 |
| **먼지** 🐈 | 집안 | 고양이 (분석 안 함, 야옹만 함) |

## 파일 구성

| 파일 | 역할 |
|---|---|
| `index.html` | 메인 화면 (더블클릭 실행) |
| `tickers.js` | **대상 종목 단일 소스** — 종목 추가/삭제는 여기만 고친다. `update_prices.py`와 `index.html`이 함께 읽음 |
| `data.js` | 시세 스냅샷(종목 + 코스피/코스닥 지수 `indices`) — `update_prices.py`가 자동 생성 |
| `analysis.js` | AI 분석 결과 (5인 + 목표가 + 상세 리포트) — LIVE 배지. 종목별 `base`(판단 당시 주가)·`baseAt`(그 주가 시점)·`updated`(분석 시각) 포함. `market` 키에 코스피/코스닥 시장 분석(화면 상단 📊 박스) |
| `history.js` | 판단 히스토리 — `archive_analysis.py`가 재분석마다 누적. 화면의 신뢰도 추이·적중 채점에 사용 |
| `archive_analysis.py` | 현재 `analysis.js`의 CHIEF 판단을 `history.js`에 기록으로 쌓음(정확히 같은 시각 재실행만 덮어씀, 하루 여러 번은 각각 보존) |
| `price_history.js` | 종목별 **실제 종가**(판단과 무관) — 5거래일 = 1페이지로 누적. 화면의 "🗓 최근 종가"에 사용 |
| `update_price_history.py` | 네이버에서 일봉 종가를 받아 `price_history.js`에 병합(중복 없음, 페이지가 5일 차면 다음 페이지) |
| `update_prices.py` / `시세갱신.command` | 시세 수동 갱신 |
| `daily_update.command` | 시세 + AI분석 통합 갱신 (cron이 자동 실행) |
| `.claude/skills/종목분석 스킬/SKILL.md` | **분석 절차의 단일 원본** — 재분석 방법·기준가 무결성 철칙·검증 절차 ("종목분석 스킬"로 호출) |
| `analysis_prompt.txt` | (구) 위 SKILL.md를 가리키는 포인터만 남김 |
| `.claude/agents/*.md` | 서브에이전트 정의 |

## 🔄 자동 갱신 (cron 등록 완료)

- **시세**: 평일 장중 **20분마다** 자동 갱신 (로그인 불필요)
- **AI 분석 + 시세**: 평일 **15:50 장마감 후** `daily_update.command` 자동 실행
- 실행 로그: `update.log`

### ⚠️ AI 자동 분석을 켜려면 (1회만)
헤드리스 분석은 Claude CLI 로그인이 필요합니다. **터미널에서 한 번만** 실행하세요:
```bash
claude login
```
로그인 후에는 매 거래일 장마감 후 팀이 자동으로 대상 종목 전체(tickers.js 기준)를 재분석해 `analysis.js`를 갱신합니다.
(로그인 전에도 시세는 계속 자동 갱신되고, 분석은 아래 수동 방법으로 갱신 가능)

### 참고
- cron은 그 시간에 **맥이 켜져 있어야** 실행됩니다.
- macOS가 cron의 데스크탑 접근을 막으면 `update.log`에 `Operation not permitted`가 남습니다 → 시스템 설정 → 개인정보 보호 및 보안 → 전체 디스크 접근에서 `cron` 허용.
- 자동 분석을 끄려면 터미널에서 `crontab -e` 후 해당 줄 삭제(또는 `crontab -r`로 전체 해제).

## 수동 갱신
- 시세만: `시세갱신.command` 더블클릭
- 시세+분석: `daily_update.command` 더블클릭 (로그인 시)
- 또는 이 폴더에서 Claude Code에게 "○○종목 다시 분석해서 analysis.js 갱신해줘"

## 화면 기능
- 종목명 옆 **대형 현재가** + PER/PBR/ROE/EPS/배당/시총/52주 지표
- **낮/밤 전환**(☀️/🌙, 90초 자동) · **효과음**(🔊)
- 캐릭터 인사·커피타임·스트레칭·고양이 쓰다듬기, 눈 깜빡임/집중 표정
- **시장 무드 반응**: 오늘 추적 종목 평균 등락으로 폭락/하락/보합/상승/급등 판정 → 상단 무드 배지 + 방 조명 색조(폭락 붉게·급등 초록빛) + 캐릭터 표정(걱정/미소)·움직임(폭락 땐 분주)·대사("가즈아!"/"물렸다…")가 달라짐
- 분석 시 각자 자리로 걸어가 분석 → 4인 카드 + CHIEF 종합(목표가·상세 리포트)
- 분석 결과 아래 **🗓 최근 종가**(실제 종가, 5거래일 단위 페이지 · ‹ › 로 과거 페이지 탐색) + **📈 판단 히스토리**: 콜/신뢰도 추이 그래프 + 적중 채점(재분석 누적) — 실제 가격 흐름과 팀 판단을 나란히 비교
- **내 포트폴리오 탭**: 보유수량·평단 입력 → 종목별 평가손익 + 총 손익 요약(오늘 손익·연 예상 배당·계좌 무드 이모지) + 팀 콜을 내 평단 기준으로 해석(익절 고려/손절 검토 등) + 팀 목표가까지 거리 + 종목별 메모. 브라우저에 프로필별로 저장(현재 프로필 「나」 1인용, 다중 사용자 확장 대비 구조)
- **애널리스트 리더보드 탭**: 4인 분석가(TARO·DIANA·NOVA·FLOW)의 강세/약세 판단이 이후 주가와 맞았는지로 적중률을 집계해 순위·MVP 표시. history.js에 쌓인 종목별 stance 기록 기반(재분석·주가 변동이 쌓일수록 정확해짐)
- **월간 캘린더 탭**: 달력에 날짜별 팀 분위기(BUY/SELL 우세)를 색으로 + 종목 주요 일정(실적발표·상장 등, NOVA가 뉴스에서 수집한 `events`) 표시. 상단에 월간 성적표(분석 횟수·팀 적중률·MVP·최고/아쉬운 콜·내 포트폴리오). 날짜 미정 이슈는 하단 목록. ‹ › 로 월 이동
- **"분석 중" 표시**: 분석 시작 시 씬 위에 귀여운 이모티콘(순환)이 뜨고 완료 시 사라짐
- **📊 코스피/코스닥 + 시장 분석**: 헤더 아래 지수 스트립(시세갱신마다 자동 최신화) + 재분석 때 함께 갱신되는 시장 국면 분석 박스
- **캘린더 일정 팝업**: 일정 있는 날짜를 터치하면 그날 일정 전체 + 팀 판단(BUY/SELL 우세)을 팝업으로 표시 (모바일 대응)
- **🔧 관리자 편집 모드**: 제목(Gaeo)을 3초 안에 5번 터치(또는 주소 뒤 `#admin`) → 비밀번호 입력 → 제목·소개글·하단 안내문 직접 수정 + 행간/글자크기 조절. 저장 시 그 브라우저(localStorage)에만 유지 — 모두에게 반영하려면 Claude에게 파일 반영 요청

## 🛡 견고성
- 시세 수집 실패 시 **이전 값 유지 + `⚠️ 시세 지연` 표시**(사라지거나 옛 하드코딩 값으로 조용히 바뀌지 않음)
